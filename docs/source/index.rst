Welcome to the documentation for **Libhankel**.

LibHankel is a C library with a number of methods for computing Hankel transforms, re-implemented based on the strategies available in [SASfit](https://github.com/SASfit/SASfit). The methods are effectively different ways of approximating the integral that constitutes the Hankel transform.
