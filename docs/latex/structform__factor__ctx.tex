\doxysubsection{Detailed Description}
Context structure for form factor callbacks. 

This structure is used to pass user-\/defined data to functions of type \mbox{\hyperlink{libhankel_8h_a0e1024037ef107f5a9e37d6932af02c2}{form\+\_\+factor\+\_\+f}}. The contents of the context are managed by the user and interpreted by the callback implementation.
