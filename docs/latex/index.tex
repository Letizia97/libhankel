.. \+\_\+overview\+:

Welcome to the documentation for {\bfseries{Libhankel}}.

Lib\+Hankel is a C library with a number of methods for computing Hankel transforms, re-\/implemented based on the strategies available in \href{https://github.com/SASfit/SASfit}{\texttt{ SASfit}}. The methods are effectively different ways of approximating the integral that constitutes the Hankel transform.

Please head to \char`\"{}\+Related Pages\char`\"{} in the above toolbar to find more information on how to install and use the library.
