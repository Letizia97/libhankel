 \hypertarget{usage_autotoc_md0}{}\doxysection{Main function}\label{usage_autotoc_md0}
The main function a user will be using is {\ttfamily \mbox{\hyperlink{libhankel_8h_a482666838880b9e0b91a12955a3b335b}{hankel\+\_\+transform()}}}. Please click on the name to find out what parameters the user receives as well as their types. The parameter {\ttfamily strategy\+\_\+name} essentially allows the user the select one of the 14 Hankel transform strategies available. This further explained under \char`\"{}\+Note\char`\"{} at {\ttfamily \mbox{\hyperlink{libhankel_8h_a482666838880b9e0b91a12955a3b335b}{hankel\+\_\+transform()}}}.\hypertarget{usage_autotoc_md1}{}\doxysection{Status codes}\label{usage_autotoc_md1}
