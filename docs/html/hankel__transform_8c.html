Computes Hankel transform using the method specified by the user.

Note
The parameter strategy_name can be any of "QWE_Chave", "QWE_Key", "DHT_6", "DHT_7", "DHT_8", "DHT_9", "DHT_10", "DHT_11".
Parameters
nuorder of bessel function - must be 0 or 1
fpointer to form factor function
xvalue at which to compute the transform
f_paramsparams for form factor
outputpointer to var containing output from transform
strategy_namestr corresponding to the strategy name
strategy_paramsstruct containing params for the specific strategy (please use the docs to find params required by each strategy)
